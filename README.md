## Roboti dokumentatsioon

Meie robot on lahe.
Roboti alus on 3mm akrüül
keeramiseks kasutame servo mootorit (teada pole selle)
robootika musta ja kolast värvi rattad (vahetamise ära ratta kummid et velgesid kasutada mis sobib mootori ülekandele paremini)
elektroonika on roboti küljes velkroga
kaamera takistuste nägemiseks - huskylens kaamera (kaseiga)
Seinte tuvastamiseks ultra sonic andurid (kaseiga)


energy source - Litium patareid
